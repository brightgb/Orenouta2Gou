キャッシュで画像サンプル
ルーティング
Route::get('/index/testImage/{file}/{image}/cache', 'Controller@testTop');
本コントローラ（キャッシュはAdminファイルになる）
$image = route('sukebo.test.top', ['file'=>'Admin','image'=>'test']);
別コントローラ
public function testTop($file, $image)
{
    $path = $file.'/'.$image.'.jpg';
    $imgProc = new ImageProc();  // このメソッドで、/storage/app/が付く
    return $imgProc->imgPublicCache($path, 'l')->response();
}
